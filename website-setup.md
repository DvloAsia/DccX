# GitHub Pages 配置步骤

## 网站已上传到：
- **仓库**: https://github.com/DvloAsia/DccXWeb
- **网站文件**: 双语网站，中英文版本

## 启用 GitHub Pages：
1. 访问 https://github.com/DvloAsia/DccXWeb/settings/pages
2. 选择 **Source**: GitHub Actions
3. 保存设置

## GitHub Pages 会自动部署
- **部署后地址**: https://DvloAsia.github.io/DccXWeb/
- **中文版**: https://DvloAsia.github.io/DccXWeb/zh/
- **英文版**: https://DvloAsia.github.io/DccXWeb/en/

## 网站特点：
✅ **双语支持**: 中文和英文切换
✅ **响应式设计**: 适配手机、平板、电脑
✅ **美观 UI**: 现代化设计，蓝色主题
✅ **简化域名**: 更简洁的网址
✅ **语言切换**: 右上角语言按钮

## 测试网站：
1. 等待 GitHub Pages 部署完成（通常几分钟）
2. 访问：https://DvloAsia.github.io/DccXWeb/
3. 测试语言切换功能
4. 测试各设备显示效果

## 备用方案（如果 GitHub Pages 配置问题）：
1. 手动在仓库 Settings 中配置 Pages
2. 选择 Branch: main, Folder: /
3. 或者选择 Branch: main, Folder: /website
