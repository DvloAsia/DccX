安装指南

Termux 安装

pkg update pkg install git gcc make git clone https://github.com/dvlo/DccX cd DccX make ./dccx examples/test.dvx

Linux 安装

sudo apt update sudo apt install git gcc make git clone https://github.com/dvlo/DccX cd DccX make sudo make install

Windows 安装

Windows 用户需要安装 MinGW 环境或使用预编译的二进制文件。

快速开始

第一个 DccX 程序

# 第一个 DccX 程序 func main() { # 打印 Hello World print "Hello, DccX!" # 执行 Shell 命令 shell "ls -la" # 变量声明 name = "Dvlo" age = 25 # 条件判断 if age > 18 { print "成年人" } }

运行程序

dccx program.dvx

编译程序

dccx -c program.dvx -o program.exe

基本语法

DccX 语法融合了多种编程语言的特性。

注释

# 这是单行注释（Shell 风格） // 这也是单行注释（C 风格） /* 这是多行注释 */

语句分隔符

每条语句以换行结束，不需要额外的分隔符。

代码块

if condition { # 代码块使用大括号 print "条件成立" }

变量和类型

变量声明

# 类型声明 num x = 10 # 数字类型 str text = "Hello" # 字符串类型 bool flag = true # 布尔类型 # 类型推断 var y = 20 # 自动推断为数字 var message = "World" # 自动推断为字符串

数据类型

类型 描述 示例
num 数字（整数/浮点数） num x = 10.5
str 字符串 str s = "Hello"
bool 布尔值 bool b = true
list 列表/数组 list arr = [1, 2, 3]
dict 字典/哈希表 dict m = {"key": "value"}
func 函数引用 func f = add
file 文件对象 file f = open("file.txt")
proc 进程对象 proc p = spawn("ls")

函数定义

基本函数

func add(a, b) { return a + b } func greet(name: str): str { return "Hello, " + name }

匿名函数

var square = func(x) { return x * x } # Lambda 表达式 var sum = (a, b) => a + b

默认参数

func multiply(a, b = 1) { return a * b }

高阶函数

func map(func, list) { result = [] for item in list { result.append(func(item)) } return result } # 使用高阶函数 numbers = [1, 2, 3, 4] squared = map(square, numbers)

控制结构

条件语句

if condition { # 代码块 } elseif condition { # 代码块 } else { # 代码块 }

循环结构

# for 循环 for i in range(0, 10) { print i } # C 风格 for 循环 for i = 0; i < 10; i++ { print i } # while 循环 while condition { # 循环体 } # Shell 风格循环 for file in $(ls *.dvx) { print "File: " + file }

Switch 语句

switch value { case 1: print "One" case 2: print "Two" default: print "Other" }
